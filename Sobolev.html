Relationship between Sobolev spaces

In this webpage, we assume \( \Omega\) to be a Lipschitz domain.

Definitions

Definition 1 \[ H^{-s} (\mathbb{R}^n ) := \left[ H^s(\mathbb{R}^n )\right]^*,\quad \text{ for } s\in \mathbb{R}. \] Definition 2 For any closed set \( F\subset \mathbb{R}^n\), we define the associated Sobolev space of order \( s\) by \[ H_F^s := \left\{ u\in H^s(\mathbb{R}^n \, :\, \mathop{\mathrm{supp}} u\subset F\right\}. \] Definition 3 For any non-empty open set \( \Omega\subset \mathbb{R}^n\) we define \[ H^s (\Omega ) := \left\{ u\in \mathcal{D}^*(\Omega ): u = U|_\Omega \text{ for some } U\in H^s(\mathbb{R}^n )\right\}. \] Definition 4 \[ \tilde{H}^s (\Omega ) := \text{ closure of } \mathcal{D}(\Omega )\text{ in } H^s(\mathbb{R}^n). \] Definition 5 \[ H^s_0 (\Omega ) := \text{ closure of } \mathcal{D}(\Omega )\text{ in } H^s(\Omega ). \]

Theorems

Theorem 3.30 \( H^s(\Omega )^* = \tilde{H}^{-s}(\Omega )\) and \( \tilde{H}^s(\Omega)^* = H^{-s}(\Omega )\) for all \( s\in \mathbb{R}\).
Theorem 3.29 \( \tilde{H}^s(\Omega ) = H_{\overline{\Omega}}^s\) for \( s\in \mathbb{R}\).
Theorem 3.33 Let \( s\geq 0\) and \( s\not\in \left\{ {1\over 2}, {3\over 2}, {5\over 2}, \ldots \right\}\), then \[ \tilde{H}^s (\Omega ) = H_0^s(\Omega ). \] Theorem 3.40 \( H_0^s(\Omega ) = H^s(\Omega )\) for \( 0\leq s\leq 1/2\).

Reference

McLean, W.C.H., 2000. Strongly elliptic systems and boundary integral equations. Cambridge university press.
