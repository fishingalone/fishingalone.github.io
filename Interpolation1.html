Interpolation of Soblev spaces

If \( X_0\) and \( X_1\) are Sobolev spaces, then the K-functional can be computed explicitly via the Fourier transform.
Theorem 1 Suppose \( s_0,s_1\in \mathbb{R}\). Let \( s=(1-\theta )s_0 + \theta s_1, 0<\theta <1 \) be a number in between. We can prove the following interpolation result \[ \left( H^{s_0}(\mathbb{R}^n), H^{s_1}(\mathbb{R}^n) \right)_{\theta,2} = H^s(\mathbb{R}^n). \]

Let \( u=u_0+u_1\) with \( u_j \in H^{s_j}(\mathbb{R}^n)\) for \( j=0,1\). We write the Sobolev norm using the Fourier definition: \[ \| u_0\|_{H^{s_0}(\mathbb{R}^n)}^2 + t^2\| u_1\|_{H^{s_1}(\mathbb{R}^n)}^2 = \int_{\mathbb{R}^n} (1+|\xi|^2)^{s_0}|\hat{u}_0(\xi)|^2 + t^2(1+|\xi|^2)^{s_1} |\hat{u}_1(\xi)|^2 \mathrm{d}\xi. \] Since \( \hat{u}(\xi) = \hat{u}_0(\xi ) + \hat{u}_1(\xi )\), we obtain the minimum of the integrand from B.4 that for each \( \xi\). Therefore \[ \begin{align*} K (t,u;H^{s_0}(\mathbb{R}^n) ,H^{s_1}(\mathbb{R}^n) )^2 &= \int_{\mathbb{R}^n} {t^2(1+|\xi|^2)^{s_0+s_1}\over (1+|\xi|^2)^{s_0} + t^2(1+|\xi|^2)^{s_1}} |\hat{u}(\xi)|^2 \,\mathrm{d} \xi \\ & = \int_{\mathbb{R}^n} (1+|\xi|^2)^{s_0} f(a(\xi) t)^2 |\hat{u}(\xi)|^2\,\mathrm{d} \xi, \end{align*} \] where \[ a(\xi ) = (1+|\xi|^2)^{(s_1-s_0)/2}\quad \text{ and } \quad f(t) = {t\over \sqrt{1+t^2}}. \] Thus, \[ \| K(\cdot, u)\|_{\theta,2}^2 = \int_{\mathbb{R}^n} (1+|\xi|^2)^{s_0} a(\xi)^{2\theta} \| f\|_{\theta,2}^2 |\hat{u}(\xi )|^2\,\mathrm{d} \xi = \| f\|_{\theta,2}^2 \| u\|_{H^s(\mathbb{R}^n)}^2. \] Finally, the norm\( \| \cdot\)_{K_{\theta,2}(X)}\) is equivalent to the norm \( \| \cdot\|_{H^s(\mathbb{R}^n)}\).

B.4 Fix real numbers \( A_0>0\) and \( A_1>0\) and a complex number \( z\). Show that \[ \min_{z=z_0+z_1} \left( A_0 |z_0|^2 + A_1 |z_1|^2 \right) = {A_0A_1\over A_0+A_1} |z|^2, \] and that the minimum is achieved when \( A_0z_0 = A_1z_1 = A_0A_1z/(A_0+A_1)\).

We write \( z_1 = z - z_0\) and use the cosine rule, \[ \begin{align*} \left( A_0 |z_0|^2 + A_1 |z_1|^2 \right) & = A_0 |z_0|^2 + A_1 |z-z_0|^2 \\ & = A_0 |z_0|^2 + A_1\left( |z|^2 +|z_0|^2 - 2|z||z_0|\cos(z,z_0)\right) \\ & = A_1 |z|^2 + (A_0+A_1) |z_0|^2 - 2|z||z_0|\cos(z,z_0). \end{align*} \] This can only be minimized by choosing \( z_0\) to be parallel to \( z\) so that \( \cos(z,z_0) = 1\). Set \( z_0 = cz\) for some positive constant \( c>0\). Solve for \( c\) so that the equation is minimized.
